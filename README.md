# 简介  

根据密码字典，尝试猜测SSH服务器密码。  

# 约束  

使用本工具时，请注意对端服务器的安全配置。如果对端服务器的安全配置较高，连接出错若干次，会导致服务器禁止登陆若干时间。在此情况下，请不要使用本工具。  

# 使用  

上传此工具到Linux环境的任意目录，执行`bash ssh_password_cracker.sh <ip> <user>`。
